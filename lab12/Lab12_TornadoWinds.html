Atms 305 Lab 12 - Tornado winds - Fall 2024

Perhaps the simplest mathematical model of tornado winds is the Rankine vortex, sometimes called the Rankine Combined Vortex, dating back to 1858. It features a rigid core (solid-body!) inner vortex in which swirling (tangential) wind increases linearly with radius, surrounded by a gradually weakening (as 1/R) wind field.

In this exercise you will create a Rankine vortex tangential wind field, print the values and plot the result. This problem utilizes NumPy and Matplotlib.

# >> E: LOOP, FILL IN "VT" ARRAY
# Use a for-loop to step through all values in your radius array.
#   The discusson below assumes your for-loop variable is "n".
# INSIDE the loop, set the tangential velocity Vt for each array element like so:
#   IF the radius radius[n] for *that element* is less than RMW:
#       set Vt[n] to Vmax*radius[n]/RMW
#   Otherwise -
#       set Vt[n] to Vmax*RMW/radius[n]
#
#   Finally, still in the loop, but OUTSIDE any if-else statements:
#     print out the radius and tangential wind for that element.
#   With the print inside a loop, you should get a list of values.
# *Format* your output like this:  (also to check your code)
#           For radius (m) =  40, Vt (m/s) =  58.67
#
#  (use 3 digits for radius -> whole numbers! - and format Vt as XXX.YY)
for n in range(len(radius)):
    if Vt[n] < RMW:
        Vt[n] = Vmax*radius[n]/RMW
    else:
        Vt[n] = Vmax*RMW/radius[n]
    print('For radius (m) = {:3}, Vt (m/s) = {:.2f}'.format(radius[n], Vt[n]))
